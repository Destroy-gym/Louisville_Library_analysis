# Louisville_Library_analysis

This file is a look into Louisville's Free Public Library (LFPL). This was intended to look into the library collections and demographic data for Louisville Kentucky (Jefferson County)


# How to use 

## How to Use
1. Clone this repository.
2. Install the required Python packages:  
   pip install -r requirements.txt
3. Open `library_analysis.ipynb` in Jupyter Notebook or JupyterLab.

## Example Output
The analysis shows the distribution of literature between branches. With the branch on Main being the largest repository.
